# دليل صفحات الموقع العامة — VetPedia

> منصة المعرفة البيطرية: الأمراض، الأدوية، المواد الفعالة، والشركات

---

## 1. الصفحة الرئيسية (`/`)

**المتحكم:** `HomeController`

**الوصف:** الصفحة الرئيسية للموقع. تعرض نظرة عامة على المنصة مع:
- شعار المنصة وجملة تعريفية
- شريط بحث سريع (نموذج بحث)
- إحصائيات عامة: عدد المنتجات والأمراض والشركات والمواد الفعالة (كل منها كارد قابل للنقر)
- أحدث المنتجات المضافة (أقصى 5)
- أحدث الشركات المضافة (أقصى 5)

**الاستخدام:** نقطة الدخول الرئيسية للزوار. يستعرض حجم قاعدة البيانات ويوجه إلى الأقسام المختلفة.

---

## 2. المنتجات — قائمة (`/products`)

**المتحكم:** `ProductController@index`

**الوصف:** دليل المنتجات البيطرية مع جدول يعرض:
- اسم المنتج التجاري
- الشركة المصنعة
- الشكل الصيدلاني
- زر التفاصيل

**الفلاتر والبحث:**
- `search` — بحث نصي في: اسم المنتج، المواد الفعالة، الأمراض المرتبطة، الشركات
- `company` — تصفية حسب الشركة (قائمة منسدلة)
- `dosage_form` — تصفية حسب الشكل الصيدلاني
- `sort` — ترتيب: `latest` (افتراضي)، `name` (أبجدي)، `oldest` (الأقدم)

**الاستخدام:** تصفح جميع المنتجات البيطرية المسجلة في قاعدة البيانات مع إمكانية البحث والتصفية والترتيب.

---

## 3. المنتجات — عرض تفصيلي (`/products/{product:slug}`)

**المتحكم:** `ProductController@show`

**الوصف:** صفحة تفاصيل كاملة لمنتج بيطري مع:
- الاسم التجاري (بالعربية والإنجليزية)
- الشكل الصيدلاني، نوع المنتج، العبوة
- الشركة المصنعة (كرابط لصفحة الشركة)
- **الجرعات** — جدول حسب النوع الحيواني (النوع، الجرعة، طريقة الإعطاء، المدة)
- **فترات السحب** — جدول حسب النوع الحيواني (اللحم، الحليب، البيض)
- **المواد الفعالة** — قائمة بالتركيز والوحدة (كل منها رابط)
- **الأمراض المرتبطة** (روابط)
- **الشركات** — مع الدور (manufacturer / agent / distributor)
- **منتجات ذات صلة** — منتجات بنفس الأمراض (حد 10)
- **منتجات ذات صلة بالمادة الفعالة** — منتجات بنفس المواد الفعالة (حد 10)

**الاستخدام:** الاطلاع على المعلومات الكاملة لأي منتج بيطري، بما في ذلك الجرعات والفترات السحب والروابط إلى الكيانات الأخرى.

---

## 4. الشركات — قائمة (`/companies`)

**المتحكم:** `CompanyController@index`

**الوصف:** دليل الشركات البيطرية مع جدول يعرض:
- اسم الشركة
- النوع (manufacturer / agent / distributor)
- الدولة
- عدد المنتجات
- زر عرض

**البحث:** `search` — بحث في الاسم (عربي/إنجليزي) والدولة

**الاستخدام:** تصفح الشركات البيطرية المصنعة والموزعة والوكلاء.

---

## 5. الشركات — عرض تفصيلي (`/companies/{company:slug}`)

**المتحكم:** `CompanyController@show`

**الوصف:** صفحة تفاصيل شركة بيطرية مع:
- الاسم (عربي/إنجليزي)
- النوع والدولة
- **معلومات الاتصال:** هاتف، موبايل، واتساب، إيميل، موقع إلكتروني، المحافظة، منطقة التغطية، العنوان
- **قائمة المنتجات** التابعة للشركة (كل منها رابط لصفحة المنتج)
- الشركة الأم (إن وجدت)
- الشركات التابعة (إن وجدت)

**الاستخدام:** الاطلاع على ملف الشركة الكامل ومنتجاتها.

---

## 6. الأمراض — قائمة (`/diseases`)

**المتحكم:** `DiseaseController@index`

**الوصف:** دليل الأمراض البيطرية مع جدول يعرض:
- اسم المرض
- عدد المنتجات المرتبطة
- زر عرض

**البحث:** `search` — بحث في الاسم (عربي/إنجليزي)

**الاستخدام:** تصفح الأمراض البيطرية المسجلة في قاعدة البيانات.

---

## 7. الأمراض — عرض تفصيلي (`/diseases/{disease:slug}`)

**المتحكم:** `DiseaseController@show`

**الوصف:** صفحة تفاصيل مرض بيطري مع:
- الاسم (عربي/إنجليزي)
- الوصف
- عدد المنتجات المرتبطة
- عدد المواد الفعالة المرتبطة (محسوب عبر المنتجات)
- **جدول المنتجات المرتبطة:** الاسم التجاري، الشكل الصيدلاني، الشركة المصنعة، رابط للعرض
- **المواد الفعالة المرتبطة** (في الشريط الجانبي)

**الاستخدام:** الاطلاع على معلومات المرض والمنتجات والمواد الفعالة المرتبطة به.

---

## 8. المواد الفعالة — قائمة (`/active-ingredients`)

**المتحكم:** `ActiveIngredientController@index`

**الوصف:** دليل المواد الفعالة مع جدول يعرض:
- اسم المادة الفعالة
- عدد المنتجات المحتوية عليها
- زر عرض

**الاستخدام:** تصفح المواد الفعالة الدوائية البيطرية.

---

## 9. المواد الفعالة — عرض تفصيلي (`/active-ingredients/{activeIngredient:slug}`)

**المتحكم:** `ActiveIngredientController@show`

**الوصف:** صفحة تفاصيل مادة فعالة مع:
- الاسم (عربي/إنجليزي)
- الوصف
- عدد المنتجات المحتوية عليها
- عدد التصنيفات الدوائية
- **الاستطبابات** (إن وجدت)
- **موانع الاستخدام** (إن وجدت)
- **الاحتياطات** (إن وجدت)
- **الآثار الجانبية** (إن وجدت)
- **جدول المنتجات المحتوية عليها:** الاسم التجاري، التركيز/الوحدة، الشركة المصنعة، رابط للعرض
- **التصنيفات الدوائية** (في الشريط الجانبي)
- **الأمراض المرتبطة** (في الشريط الجانبي — محسوبة عبر المنتجات)

**الاستخدام:** الاطلاع على المعلومات الدوائية الكاملة للمادة الفعالة والمنتجات المحتوية عليها والأمراض المرتبطة بها.

---

## 10. البحث (`/search`)

**المتحكم:** `SearchController@index`

**الوصف:** صفحة بحث موحدة تبحث في أربعة كيانات في وقت واحد:
- `q` — نص البحث
- النتائج مقسمة إلى أربعة أقسام: المنتجات، الشركات، الأمراض، المواد الفعالة (حد 20 نتيجة لكل قسم)

**الاستخدام:** بحث سريع وشامل عبر قاعدة البيانات بأكملها.

**المتحكم:** `SitemapController`

**الوصف:** ملف XML لخوارزميات البحث (SEO). يحتوي على روابط لجميع صفحات العرض التفصيلي (المنتجات، الأمراض، الشركات، المواد الفعالة).

**الوصف:** ملف robots.txt يسمح بفهرسة جميع الصفحات ويشير إلى ملف خريطة الموقع.
